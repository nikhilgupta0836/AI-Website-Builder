# 🌐 AI Website Builder

**AI Website Builder** is a modern platform that allows anyone to create fully functional websites in minutes with the power of **AI**. Designed with the **MERN stack** and integrated with **DeepSek**, it simplifies website creation by automatically generating content, layouts, and designs.  

---

## 🚀 Features

- **AI-Powered Website Generation** – Generate complete websites with smart AI suggestions.  
- **User Authentication** – Secure login using **Firebase Google Auth**.  
- **Interactive Dashboard** – Manage multiple websites, preview changes, and track updates.  
- **Responsive & Modern Design** – Works flawlessly across desktop, tablet, and mobile devices.  
- **Payment Integration** – Stripe-based subscription for premium features.  
- **Fast & Intuitive** – Focus on your content while AI handles layouts and structure.  

---

## 🛠 Tech Stack

- **Front-End:** React, TailwindCSS  
- **Back-End:** Node.js, Express.js, MongoDB  
- **AI Services:** DeepSek API for content generation  
- **Authentication:** Firebase Authentication  
- **Payments:** Stripe  
- **Hosting:** Render / Vercel  

---

## 🌟 Why Use AI Website Builder?

Building a website from scratch can be time-consuming and technically challenging. This platform lets you:  

- Skip repetitive design and coding tasks.  
- Generate content and layouts automatically with AI.  
- Focus on growing your business or project instead of worrying about development.  
- Quickly launch professional websites without prior coding knowledge.
